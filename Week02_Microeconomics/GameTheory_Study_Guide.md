## 📚 Game Theory — Core Concepts

### Hand Notes Template

```
Game Theory：研究策略互動（非單獨決策）
Prisoner's Dilemma：兩人同時決策，合作 vs 背叛 → 雙輸均衡
Nash Equilibrium：每個玩家都係最佳反應，無人想單方面改變
Dominant Strategy：無論對手點做都係最好選擇
Repeated Game：長期合作可能出現（Tit-for-Tat 策略）
Payoff Matrix：用表格顯示每個玩家收益
```

### Case 1: 香港地產商 (New World / CK Asset / Henderson)
- 寡頭市場（少數大玩家）
- Pricing Game：同時決定樓價 → Nash Equilibrium = 高價均衡（但合作更賺）
- 結果：互相跟價 → 樓價長期高企

### Case 2: 香港銀行 (HSBC / Hang Seng / BOC)
- Interest Rate Game：減息競爭 → Prisoner's Dilemma（大家都減息 → 利潤↓）
- Dominant Strategy：跟隨減息

### Case 3: 電訊商 (CMHK / 3HK / Smartone)
- 5G 計劃價格戰 → Repeated Game：短期背叛（減價搶客），長期 Tit-for-Tat 回復合作
